## 소개

**`linux-insides`**는 리눅스 커널과 시스템의 동작 원리를 학습하고 실습한 내용을 정리한 저장소입니다.  
커널 빌드부터 시스템 콜 작성, 프로세스 관리, 메모리 구조 분석까지 **리눅스의 안쪽을 파고드는 실습 중심**의 저장소입니다.
